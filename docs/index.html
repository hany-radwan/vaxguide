VaxGuide

CLI-based Vaccination Guidance Tool

About the Project

VaxGuide is a Rust-based Command Line Interface (CLI) tool that provides general vaccination guidance based on user input. It follows public health recommendations within Germany and the EU context.

Features

  • ✔ Interactive CLI questions
  • ✔ COVID-19 & Influenza guidance
  • ✔ Pregnancy considerations
  • ✔ Bavaria-specific FSME recommendation
  • ✔ Public health–oriented logic

Technical Stack

Language: Rust

Architecture: Linux (x86_64 & aarch64)

CI/CD: GitHub Actions
